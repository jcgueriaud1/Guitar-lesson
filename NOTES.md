# Teaching Notes

Preferences and working notes for how to teach this user.

## Hard constraints
- **15 min/day maximum.** Never design a session that needs more. Assume some
  days are missed — value must land inside a single short session.
- **No rushing.** The user explicitly wants foundations solid before moving on.
  Bias toward over-consolidating rather than advancing early.

## Where they are (as of 2026-06-08)
- Knows the basic open chord *shapes*; can form them.
- Weak point: *changing* between chords smoothly and in time. This is the
  current bottleneck and the focus of early lessons.
- Acoustic guitar. Wants to play fingerstyle (no pick is the goal).

## Skill arc (rough, revise freely)
1. Clean, fast open-chord changes (One Minute Changes). ← current
2. Changes in time with a steady strum (chord + rhythm together).
3. Basic Travis picking on a single chord, then across a chord change.
4. A full fingerpicked song.
5. Switching strum ↔ fingerpick within one song.
Each rung is reinforced by learning real songs, easy → advanced.

## Format notes
- Lessons are interactive HTML with built-in feedback (timers, tally counters)
  so a single short session gives an immediate, automatic feedback loop.
- Keep one thing per lesson. The 15-min budget makes this non-negotiable.
